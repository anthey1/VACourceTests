ТЕМА: В документе Заказ Покупателя в  табличной части "Товары" при выборе номенклатуры с типом "услуга" поле "Количество" недоступно для редактирования или изменения При этом значение в поле "Количество" в ЛЮБОМ СЛУЧАЕ учитывается в итоговом количестве ТОВАРОВ по документу.

4. База Файловая

5.  Поле "Количество" недоступно для редактирования или изменения только в случае добавления в табличную часть "Товары" Номенклатуры с типом "Услуга". При добавлении такой Номенклатуры, количество, которое было указано в форме подбора (или количество по умолчанию = 1) В ЛЮБОМ СЛУЧАЕ учитывается в итоговом количестве по документу. Например, если в Заказе есть "Номенклатура_Товар" с количеством 5, в данный Заказ добавляется "Номенклатура_Услуга" с количеством 1, то в итоговое количество ТОВАРОВ по документу равно "6" (в должно быть "5" ). Так же изменить количество "Номенклатуры_Услуга" в Табличной Части "Товары" невозможно, т.к. реквизит "Количество" не доступен для редактирования в строке "Номенклатура_Услуга", что является ошибкой.

 6. Шаги для воспроизведения
  - Создаем Заказ Покупателя.
  - Добавляем в Заказ товар "Товар без характеристик" с количеством = "5".
  - Добавляем в Заказ товар "Услуга" с количеством = "1".
  - Проверяем Общее количество ТОВАРОВ по Заказу = "6".
  - Пробуем изменить количество в строке "Товар без характеристик".
  - Пробуем изменить количество в строке "Услуга".

7. Фактический результат: После добавления в Табличную Часть "Товары" Номенклатуры типа "Услуга" - реквизит табличной части "Количество" в строке с Номенклатурой типа "Услуга" недоступен для редактирования. Итоговое Количество "Товаров" включает и количество "Услуг".
8. Ожидаемый результат: После добавления в Табличную Часть "Товары" Номенклатуры типа "Услуга" реквизит табличной части "Количество" для строки с Номенклатурой типа "Услуга" доступен для редактирования. Итоговое Количество "Товаров" не включает количество "Услуг", а выводит итоговое количество "Услуг" в отдельном итоговом реквизите.
